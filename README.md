## Sobre

Portfólio pessoal com estética cyberpunk/dark, apresentando minha trajetória, stack técnica e os projetos que construí. Foco em frontend — interfaces, animações e experiência do usuário. Desenvolvido em HTML, CSS e JavaScript puro — sem frameworks, sem dependências.

Inclui também um **Trainer Card** interativo (React + Vite) acessível como subpágina, com navegação por abas no estilo de jogo de RPG.

## Stack do portfólio

| Camada | Tecnologia |
|--------|-----------|
| Markup | HTML5 semântico |
| Estilo | CSS3 (variáveis, grid, animações, IntersectionObserver) |
| Script | JavaScript vanilla (ES6+) |
| Trainer Card | React 18 + Vite (build estático) |
| Deploy | GitHub Pages |
